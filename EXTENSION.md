# NEXUS Extension Installation Guide

## Chrome (Recommended)

**Direct Install from Chrome Web Store:**
[![Chrome Web Store](https://img.shields.io/badge/Chrome-Get%20Extension-blue?logo=googlechrome&logoColor=white)](https://chromewebstore.google.com/detail/p-stream-extension/gnheenieicoichghfmjlpofcaebbgclh)

1. Click the link above or go to: https://chromewebstore.google.com/detail/p-stream-extension/gnheenieicoichghfmjlpofcaebbgclh
2. Click "Add to Chrome"
3. Click "Add extension" in the popup
4. The extension icon will appear in your toolbar
5. Visit NEXUS and the extension will automatically work!

---

## Firefox (Manual Install)

Firefox users need to install the extension manually:

### Step 1: Download

1. Go to the [NEXUS Extension Releases](https://github.com/p-stream/extension/releases)
2. Download the latest `.xpi` file

### Step 2: Install

1. Open Firefox
2. Go to `about:addons` in the address bar
3. Click the gear icon ⚙️
4. Select "Install Add-on From File..."
5. Select the downloaded `.xpi` file
6. Click "Add" when prompted

---

## Userscript (Alternative)

For browsers that don't support extensions, use the userscript:

### Step 1: Install a Userscript Manager

- **Violentmonkey** (recommended): [Install](https://violentmonkey.github.io/)
- **Tampermonkey**: [Install](https://www.tampermonkey.net/)
- **Greasemonkey** (Firefox only): [Install](https://www.greasespot.net/)

### Step 2: Install the Userscript

1. [Click here to install the userscript](https://raw.githubusercontent.com/p-stream/Userscript/main/p-stream.user.js)
2. Your userscript manager will prompt you to install
3. Click "Install" or "Confirm"
4. Reload NEXUS

## Troubleshooting

### Extension not working?

1. Make sure the extension is enabled for NEXUS
2. Try reloading the page
3. Open the extension popup and check the status
4. If it says "Not allowed", click "Enable" and reload

### Still having issues?

- Clear browser cache and cookies
- Disable other extensions that might interfere
- Try using the Userscript alternative
- Join our [Discord](https://discord.gg/7z6znYgrTG) for help
